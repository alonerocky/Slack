1, This project include two activites , one to show the user profile list
the other show the user profile detail
2, use recyclerview & cardview to display user profile list
3, use retrofit to download the user list json
4, use http://www.jsonschema2pojo.org/ to generate the models
5, use gson to parse the json to models


3rd party libraries
(1) Retrofit & Okhttp - Networking
(2) Gson - JSON parsing
(3) Glide - image download
(4) butterknife - view injection
